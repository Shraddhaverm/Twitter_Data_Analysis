# Twitter_Data_Analysis


Overview
This project analyzes tweets about Britney Spears from TSV files. It allows users to query the data for insights related to specific search terms. The functionality includes counting tweets by day, unique users, average likes, tweet locations, times of day for tweets, and identifying the top user posting about the term.

Features
Count of Tweets Per Day: Determine how many tweets were posted containing the search term on each day.
Unique User Count: Get the number of unique users who tweeted about the search term.
Average Likes: Calculate the average likes for tweets containing the search term.
Location Insights: Identify where the tweets originated in terms of place IDs.
Time of Day Analysis: Analyze what times of day the tweets were posted.
Top User: Find out which user posted the most tweets containing the search term.
